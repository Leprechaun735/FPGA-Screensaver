# FPGA-Screensaver

Screensaver hosted on a FPGA group project with John Danison and Matt Tolla

We used an Altera DE1 board and the VGA port to connect to our monitor. the goal was to have a picture that whenever it bounces into a wall it would make a noise and change the image.
We used Quartus II as the software to code the FPGAs through
